# 📝 譜面作成の手順 (How to Create Charts)

本アプリ内蔵のエディターで、誰でも簡単にオリジナルの譜面を作成できます。

### **1. エディターの起動**
タイトル画面（起動画面）で `この枠内で開始` を選択し、スタジオ設定画面へ進みます。そこで、画面下部にある **`譜面作成`** ボタンをクリックしてください。

### **2. 動画の読み込み**
`動画ファイルを選択` ボタンから、作成したい楽曲の映像ファイル（MP4等）を個別に読み込みます。

### **3. リアルタイム・レコーディング**
動画が再生されたら、リズムに合わせて**キーボード（A〜L、Space）を叩きます**。叩いた瞬間のタイミングが自動的に記録されます。
※ シークバーを操作して、気に入らない箇所を何度でもやり直すことができます。

### **4. データの書き出し**
作成が終わったら、エディター画面内の **`出力`** ボタンをクリックしてください。これで、作成した譜面データ（JSON形式のテキスト）がクリップボードにコピーされます。

### **5. ファイルとして保存**
テキストエディタにコピーした内容を貼り付け、以下の形式で保存してください。
- **保存先**: `assets/charts/` フォルダ内
- **記述形式**: `const CHART = [ ... ];` (貼り付けたコードをそのまま保存)
- **ファイル名例**: `asongforyou_hard.txt`
