# ArcAngel (Arc-Angel) 👼💻

![ArcAngel Favicon](assets/images/favicon.png)

**ArcAngel** は、ブラウザで動作するプレミアムな動画同期型リズムアクションゲームです。  
お手持ちの動画ファイル（MP4等）を読み込み、音楽と映像に完全に同期したリズムプレイを楽しむことができます。

---

## 🌟 主な特徴 (Features)

- **本格的な 9レーン（0〜8）リズムアクション**
  - スペースキーをセンターに据えた、左右対称でバランスの取れたプレイフィールド。
- **動画ファイル同期機能**
  - ローカルのMP4などの動画ファイルを読み込み、映像を背景としてプレイ可能。
- **内蔵譜面エディター**
  - リアルタイムでノーツを配置・編集し、自分だけの譜面を作成・保存できます。
- **プレミアム・デザイン**
  - Tailwind CSS を活用したネオン・ガラスモーフィズムな美しい UI。
- **マルチデバイス対応**
  - PC（キーボード）だけでなく、スマートフォン（全画面タッチ）でのプレイも。
- **Google Analytics (GA4) 連携**
  - プレイ開始時やリザルト表示時のカスタムイベント計測に対応。

---

## 🎮 操作方法 (How to Play)

### キーボード操作 (Keyboard)
- **左レーン (0-3)**: `A` `S` `D` `F`
- **センターレーン (4)**: `SPACE` (または `H`)
- **右レーン (5-8)**: `J` `K` `L` `;`
- **全画面切り替え**: 右上のアイコン ✥ をクリック

### スマートフォン操作 (Touch)
- 画面上の各判定ボタンを直接タップ。マルチタッチ対応。

## 🚀 導入・開発 (Technical)

### 技術構成
- **Frontend**: HTML5, Vanilla JavaScript, CSS3 (Tailwind CSS)
- **Deployment**: GitHub Pages 対応（`index.html` および `latest/` フォルダ）
- **Metrics**: Google Analytics 4 (`gtag.js`)

### ファイル構成
- `index.html`: 安定版 (Stable)
- `latest/v0.8.html`: 最新の開発版 (Development)
- `assets/charts/`: 各楽曲の譜面データ (JSON/Text)
