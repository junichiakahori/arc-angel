# 📜 Update History (CHANGELOG)

## [v0.9.0] - 2026-04-06 (Major Chart Update)
### ✨ 新曲追加 & 大規模メンテ (New Songs & Maintenace)
- **v0.9 Hotfix (2026/04/07 00:44)**:
    - 動画のロード中やバッファリング中に譜面（ノーツ）が先行して流れてしまう同期不具合を修正。
    - 動画の再生状態（`readyState`, `paused`, `seeking`）を厳密に監視し、ゲームタイマーと完全に同期するように改善。
- **v0.9 Hotfix (22:16)**:
    - 新曲「僕らは今のなかで (μ's)」の HARD 譜面のレコーディングと、3回分の平均化クリーニングが完了。正式に譜面データとして採用。
- **v0.9 Hotfix (21:44)**:
    - プレイボタン押下後、動画のシークと再生準備が完了するまで「ライブ準備中...」のロード表示を維持するように改善。
    - プログレスバーが動き出すタイミングで一気にゲーム画面へ遷移させることで、不自然な空白時間を解消。
- **v0.9 Hotfix (21:32)**:
    - リリース直後にキー入力（キーボード・タッチ）が一切反応しなくなっていた重大な不具合を修正。
    - `initGame` 関数内から消失していた `setupInput()` と `resize()` の呼び出しを復元し、初期化フローを正常化。
- **v0.9 Hotfix (19:42)**:
    - `showLoading` 関数内で ID `loading-text` が見つからず JS がクラッシュし、ゲームが進行不能になる問題を修正。
    - `updateVersionInfo` も ID 不足に備え安全な実装に変更。
- **New Feature**: プレイボタン押下後のフェードイン演出とローディング表示を追加。
- **Maintenance**: エディター画面から直接タイトルに戻れるボタンを追加、UI の各種マージンの微調整。
